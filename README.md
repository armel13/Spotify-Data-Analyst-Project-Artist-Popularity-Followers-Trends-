Spotify Song Analysis – Data Analyst Project

Repository ini berisi proyek analisis data Spotify yang dilakukan menggunakan Microsoft Excel. Dataset terdiri dari sekitar 8.700 entri lagu, mencakup informasi lengkap seperti:

Nama artis

Popularitas lagu

Jumlah pengikut artis

Genre musik

Tanggal rilis album/track

Jenis rilis (album, single, dll.)


Seluruh data telah melalui proses data cleaning dan transformasi sebelum dianalisis menggunakan PivotTable dan visualisasi berbasis PivotChart.


Fokus Analisis

Proyek ini melakukan beberapa analisis utama, yaitu:

1. Rata-rata popularitas artis

Menggunakan PivotTable untuk menghitung:

Average track popularity tiap artis

Persentase kontribusi popularitas terhadap total dataset


2. Hubungan antara popularitas dan jumlah followers

Analisis membandingkan:

Seberapa populer lagu seorang artis

Seberapa besar basis penggemarnya (artist_followers)


Tujuannya adalah memahami apakah artis dengan banyak followers otomatis memiliki lagu yang populer.

3. Tren popularitas berdasarkan tahun rilis

Menggunakan grouping tahun pada PivotTable untuk melihat:

Kenaikan/penurunan popularitas lagu per tahun

Pola perilisan musik dari waktu ke waktu


4. Distribusi genre

Filter genre digunakan untuk:

Melihat popularitas dalam kategori tertentu

Mengamati karakteristik genre dengan kontribusi tertinggi


 Metode Analisis

Analisis dilakukan dengan beberapa tools Excel:

PivotTable → untuk agregasi data

Value Field Settings (Average, % of Parent Total) → untuk perhitungan popularitas dan followers

Filter berdasarkan tahun & genre → untuk analisis yang lebih spesifik

PivotChart (opsional) → untuk visualisasi data
